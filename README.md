```markdown
# Python Data Project

Intern team project covering the full data pipeline in Python.

## Team Structure

| Team | Folder | Responsibility |
|------|--------|----------------|
| Data Sourcing | `data_sourcing/` | Collect & fetch raw data (APIs, scraping, CSV ingestion) |
| Data Processing | `data_processing/` | Clean, transform & validate data |
| Data Analytics | `data_analytics/` | Analyse, visualise & report insights |

## Getting Started
pip install -r requirements.txt

## Workflow Rules
- Each team works only inside their own folder
- All changes go through Pull Requests — no direct push to main
- PRs need at least 1 review before merging
- Use clear commit messages: feat:, fix:, docs:, refactor:
```
